## Installation
Add this line to your application's Gemfile:

```ruby
gem 'bootstrap-select-rails-turbolinks'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install bootstrap-select-rails-turbolinks
```
